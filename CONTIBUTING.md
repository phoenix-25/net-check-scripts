# Contributing to net-check-scripts

Thank you for considering contributing to this project!

## How to contribute

- Fork the repository  
- Create a feature branch (`git checkout -b feature-name`)  
- Make your changes  
- Test your changes locally  
- Submit a pull request describing your work

## Reporting issues

Please open an issue if you find a bug or want to request a feature. Include detailed information and steps to reproduce.

## Code style

- Write clear, readable code  
- Comment your code where necessary  
- Follow existing script style (Bash or Python conventions)

---

Thank you for your support!
